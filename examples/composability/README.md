# Composability Examples

Examples showing how Python configuration and reusable generators can build
larger hardware structures.

| File | Demonstrates |
| --- | --- |
| `pipeline_generator.py` | Building a pipeline from reusable stages. |
| `register_bank.py` | Generating a register bank from Python data. |
| `design_explorer.py` | Exploring design variants from Python. |

Run an example from the repository root:

```powershell
uv run python examples\composability\pipeline_generator.py
```
