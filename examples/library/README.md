# Library Examples

Examples for reusable DSL component helpers in `veriforge.dsl.lib`.

| File | Demonstrates |
| --- | --- |
| `fifo_example.py` | FIFO component construction. |
| `cdc_example.py` | Clock-domain-crossing helpers. |
| `codec_example.py` | Encoder and decoder helpers. |
| `dsp_example.py` | DSP-oriented helpers. |
| `xilinx_example.py` | Xilinx-oriented inference helpers. |

Run an example from the repository root:

```powershell
uv run python examples\library\fifo_example.py
```
