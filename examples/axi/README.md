# AXI Examples

Examples for bus-oriented DSL patterns.

| File | Demonstrates |
| --- | --- |
| `axi_stream_example.py` | AXI-Stream interface and pipeline patterns. |
| `axi_lite_example.py` | AXI4-Lite register-oriented patterns. |

Run an example from the repository root:

```powershell
uv run python examples\axi\axi_stream_example.py
```
